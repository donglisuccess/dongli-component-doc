const getData = () => {
    let params = {
        current: pageIndex.value,
        pageSize: pageSize.value
    }
    axios.post('/api/list', params).then((res: any) => {
        if(res.data.code === 200) {
            data.value = res.data.data.rows
            console.log(data.value)
            total.value = res.data.data.total
        }
    })
}

Props ​

名称描述类型默认值必填
options表格配置项TableOptions[]--是
dataList表格数据any[]--是
loading加载中效果booleanfalse否
elementLoadingText显示在加载图标下方的加载文案string--否
elementLoadingSpinner自定义加载图标string--否
elementLoadingSvg自定义加载图标string--否
elementLoadingBackground背景遮盖颜色string--否
checkIcon单元格编辑确认图标stringCheck否
closeIcon单元格编辑取消图标stringClose否
isEditRow是否整行可编辑booleanfalse否
isEditRowIndex可编辑的操作项下标string''否
pagination是否显示分页信息booleanfalse否
pageIndex分页下标number1否
pageSize每一页展示条数number10否
total总条数number0否
pageSizes分页下拉列表number[][10, 20, 30, 40]否
pageinationAlign分页菜单所处位置enum(left/center/right)right否

TableOptions 数据类型说明 ​

key说明类型
prop字段名称string
label表头名称string
width对应列的宽度string/number
align对其方式left/center/right
slot是否自定义插槽boolean
action是否存在操作项boolean
editable是否可编辑的boolean

事件 ​

事件名说明参数
confirm单元格触发编辑后确认scope
cancel单元格触发编辑后取消scope
handleSizeChange更改pageSizeval:number
handleCurrentChange更改pageIndexval:number
