图标选择 ​

用户点击按钮，弹出框显示所有按钮，点击按钮可以赋值按钮代码。

基本用法 ​


按需引入 ​

ts
import myChooseIcon from 'dongli-component-lib/chooseIcon'
import 'dongli-component-lib/chooseIcon/style.css'
// ...

const app = createApp(App)
app.use(myChooseIcon)
// ...

实例代码 ​

html
<div>
    <my-choose-icon title="选择图标" v-model:visible="visible">
        <span>选择图标</span>
    </my-choose-icon>
</div>
js
import { ref } from 'vue'
let visible = ref<boolean>(false)
