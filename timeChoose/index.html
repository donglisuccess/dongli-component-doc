时间选择 ​

提供开始时间和结束时间选择联动、开始时间和结束时间选择联动。

基本用法 ​



开始时间
结束时间

按需引入 ​

ts
import timeChoose from 'dongli-component-lib/timeChoose'
// ...

const app = createApp(App)
app.use(timeChoose)
// ...

实例代码 ​

html
    <my-time-choose
        :startOptions="startOptions"
        :endOptions="endOptions"
        @startChange="startChange"
        @endChange="endChange"
        >
    </my-time-choose>
ts
interface endTimeType {
    startTime: string;
    endTime: string
}
const startChange = (val: String) => {
    console.log("startChange:", val)
}
const endChange = (val: endTimeType) => {
    console.log("endChange:", val)
}
const startOptions = {
    clearable: false
}
const endOptions = {
    clearable: false
}

Props ​

名称描述类型默认值必填
startPlaceHolder开始的占位string开始时间否
startBeginTime开始时间string08:00否
startStep开始时间的步长string00:15否
startEndTime开始的结束时间string18:00否
endPlaceHolder结束时间占位string结束时间否
endBeginTime结束开始时间string08:00否
endStep结束步长string00:15否
endEndTime结束时间string18:00否

事件 ​

事件名说明参数
startChange开始时间改变触发val:string
endChange结束时间改变触发val:endTimeType

endTimeType 数据类型说明 ​

key说明类型
startTime开始时间string
endTime结束时间string
