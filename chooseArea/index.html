省市区选择 ​

通过下拉选择，实现省市区街道多级联动。

基本用法 ​



请选择省
请选择市
请选择区


请选择省
请选择市
请选择区
请选择街道

按需引入 ​

1、省市区按需引入

ts
import myChooseArea from 'dongli-component-lib/chooseArea'
import 'dongli-component/chooseArea/style.css'
// ...

const app = createApp(App)
app.use(myChooseArea)
// ...

2、省市区街道按需引入

ts
import myChooseAreaPlus from 'dongli-component-lib/chooseAreaPlus'
import 'dongli-component-lib/chooseAreaPlus/style.css'
// ...

const app = createApp(App)
app.use(myChooseAreaPlus)
// ...

实例代码 ​

html
<div>
    <my-choose-area @change="change"></my-choose-area>
    <my-choose-area-plus @change="changePlus"></my-choose-area-plus>
</div>

事件 ​

事件名说明参数
change选择最后一级后触发val: AreaInfo（地区数据）

key说明类型
province省份的数据val: Value
city城市的数据val:Value
area区域的数据val:Value
street街道的数据val:Value
