通知菜单 ​

提供消息通知悬浮提示内容，支持多种可选择配置。

基本用法 ​



99+

按需引入 ​

ts
import notification from 'dongli-component-lib/notification'
// ...

const app = createApp(App)
app.use(notification)
// ...

Props ​

名称描述类型默认值必填
icon图标stringBell否
value显示值string / number0否
max最大显示值number99否
isDot是否显示点booleanfalse否
popoverWidthpopover的宽度像素值：number300否

事件 ​

事件名说明参数
itemClick点击列表项触发(item:ListItemType, index: number):void
actionClick点击操作项触发(item:ActionsListType, index: number):void

ListItemType 数据类型说明 ​

key说明类型必填
avator头像string否
title名称string否
time时间string否
content内容string否
tag标签string否
tagType标签类型''/primary/success/info/warning/danger否

ActionsListType 数据类型说明 ​

key说明类型必填
icon图标string否
text操作描述文字string是

Slots ​

名称说明
reference触发 Popconfirm 显示的 HTML 元素
defaultPopconfirm 弹出框展示内容
Pager
Previous page趋势标记
Next page导航菜单
